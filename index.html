📌 事件

事件监听器
添加事件监听器-addEventListener()
-参数1表示事件，如click、mouseover、keydown
-参数2表示事件发生时调用的函数，多为匿名函数
按钮事件有foucus、blur、dblclick、mouseover、mouseout
移除事件监听器-removeEventListener()
另外两种注册事件处理程序的方法：
事件处理属性-如onclick
内联事件处理程序（不推荐）
事件对象：
有时在事件处理函数中，会看到一个参数，名称为 event、evt 或 e。被称为事件对象

🧨 事件冒泡

事件是从被点击的最内层元素冒出来的

stopPropagation()-当它在事件处理程序中被调用时，可以防止事件冒泡到其他元素
事件捕获：与事件冒泡顺序相反
事件捕获默认被禁用，要启用它，你必须在 addEventListener（） 里传递caption选项
如container.addEventListener(“click”, handleClick, { capture: true });
事件冒泡实用方法：
当用户与大量子元素中的任何一个进行交互时执行某些代码时，我们在它们的父元素上设置事件监听器，并让事件从子元素冒泡到父元素，而不是必须单独在每个子元素上设置事件监听器
target和currentTarget
target 指向事件最初被触发的元素，而 currentTarget 指向事件处理器被附加的元素

📦 对象

对象是相关数据和功能的集合，通常包含多个变量和函数（当它们位于对象内部时称为属性和方法）
const objectName = {
    member1Name: member1Value,
    function1name(){
        ...
    }
};
访问对象：
    用.访问对象的属性和方法
    用[]，objectname["memeber1Name"]
设置成员
    更新现有属性和方法的值,也可以创建全新的成员
this关键字
    this通常指当前代码正在执行的对象，在对象方法的语境中，this指的是该方法被调用的对象
构造函数
    用new关键字调用的函数

        this.name = name;
        this.introduceSelf = function () {
            console.log(`Hi! I'm ${this.name}.`);

    }
    const salva = new Person("Salva");

📄 JSON

JSON 是一种基于文本的数据格式，遵循 JavaScript 对象语法
可以把它解析成一个 JavaScript 对象，像处理用相同对象字面量语法声明的对象一样使用它
也可以将数组转换为 JSON
JSON语法
    对于原语，JSON 可以包含字符串文字、数字文字、true、false 和 null
    对于非原语，JSON 可以包含对象文字和数组，但不能包含函数或其他对象类型
    对象的每个属性名称必须是用双引号包围的字符串文字
对象与文本之间的转换
    parse（）：接受 JSON 字符串作为参数，并返回相应的 JavaScript 对象
    stringify（）：接受一个对象作为参数，并返回等价的 JSON 字符串

toggle()函数一般用于在两个对立的状态（如真/假、开/关）之间翻转

⚡ 同步编程

同步函数：当同步函数运行时不能做其他任何操作
    原因：JavaScript 程序是单线程的它一次只能做一件事：所以如果它在等待我们长时间运行的同步调用返回，它就不能做其他事情

🚀 异步编程

事件处理器（add Event Listener）也是异步函数
回调：一个传递给另一个函数的函数（不易处理嵌套，故现代异步API不常用）
承诺——异步编程基础
    链式承诺：
    const fetchPromise = fetch(
    "https://mdn.github.io/learning-area/javascript/apis/fetching-data/can-store/products.json",
    );
    fetchPromise.then((response) => {
        const jsonPromise = response.json();
        jsonPromise.then((data) => {
        console.log(data[0].name);
        });
    });
    重写：
    const fetchPromise = fetch(
    "https://mdn.github.io/learning-area/javascript/apis/fetching-data/can-store/products.json",
    );
    fetchPromise
        .then((response) => response.json())
        .then((data) => {
        console.log(data[0].name);
    });
    我们不必在第一个then()的处理程序中调用第二个then()，而是可以返回json()返回的promise，并在该返回值上调用第二个then()

    catch()方法——用于错误处理
        异步操作成功时调用传递至 then() 的处理程序，异步操作失败时调用catch()处理程序
    承诺三种状态：
        pending初始状态，操作尚未完成
        fulfilled操作成功，此时调用了 promise 的.then() 处理器
        rejected操作失败,此时调用承诺的 .catch() 处理程序
    Promise.all()方法，它接受一系列承诺，并返回一个承诺
    Promise.any()方法，实现一系列承诺中的任意一个，不在乎哪一个
    
    async：
        在函数开头添加async，使其成为异步函数
    await：
        使得代码在该点等待，直到Promise被解决，此时Promise的fulfilled值被视为返回值，或者rejected值被抛出

🌳 DOM 脚本

Web 浏览器重要部分

  • 窗口 —— Window
  • 导航器 —— Navigator
  • 文档 —— Document

DOM 树结构

  • 树中的每个条目称为一个节点（元素节点 or 文本节点）。
  • 关系术语：根节点、子节点、后裔节点、父节点、兄弟节点。

选择元素

要操作 DOM 中的元素，首先需要选中它并将该元素的引用存储在变量中。

如：const link = document.querySelector("a");

  • Document.querySelector()
  • Document.querySelectorAll()（现在最常用）
  • Document.getElementById()
  • Document.getElementsByTagName()

创建与添加节点

  • Document.createElement() —— 新建元素。
  • Node.appendChild() —— 先将引用存储在变量中，再传入变量使用，移到节点底部。
  • Document.createTextNode() —— 创建文本节点。

移除节点

  • Node.removeChild() —— 移除节点及其父节点的引用。
  • Element.remove() —— 基于自身引用删除节点。

操作 CSS 样式

  1. 获取样式表列表（过时方法）

    • 用 Document.styleSheets 获取所有附加到文档的样式表列表，它会返回一个类似数组的对象，里面有 CSSStyleSheet 的对象，然后根据需要添加或移除样式。

  2. 内联样式

    • 直接在想要动态样式的元素上添加内联样式 —— HTMLElement.style 属性。

      如：para.style.color = "white";

  3. 类名操作（推荐）

    • 在单独的样式表中写样式，并通过添加或删除类名来引用这些样式（使用元素的 classList 的 add() 方法）。
