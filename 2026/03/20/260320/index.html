js语法学习

2026/3/20 learn js

[toc]

📝 字符串

  • 模板字面量 (反向引号 `)
    • 在模板字面量内，你可以将 JavaScript 变量或表达式包裹在 ${ } 内。
    • 模板字面量可以写出跨多行的字符串。
  • **window.prompt()**：通过弹出对话框提示用户回答问题。
  • **String()**：将其参数转换为字符串。
  • **includes()**：判断一个较小的字符串是否存在于较大的字符串中。
  • **startsWith() / endsWith()**：判断字符串是以某个子字符串开始还是结束。
  • **indexOf()**：找到子串在更大字符串中的位置。
  • **slice()**：从字符串中提取子字符串。
  • 大小写与替换：
    • toLowerCase()
    • toUpperCase()
    • replace()
    • replaceAll()
  • **split()**：将字符串分割为数组。
    • 最简单的形式是取一个参数（分隔符），并将分隔符之间的子字符串作为数组中的项返回。

📦 数组 ([,,,...]) - 索引集合

增删元素

  • push()：在数组末尾添加一个或多个项目。
  • unshift()：将项目添加到数组开头。
  • pop()：移除数组中最后一个项目。
  • shift()：从数组中移除第一个项。
  • **splice()**：根据索引移除项目。
    • 第一个参数：说明从哪里开始移除项目。
    • 第二个参数：说明应移除多少项目。

遍历与转换

  • **for...of...**：访问数组每一项。
  • **map()**：对数组中的每个项目做同样的操作，最终留下包含更改后物品的数组。
    • 给 map() 赋予一个函数，它会对数组中的每个项调用该函数一次。
  • **filter()**：创建一个新的数组，只包含原始数组中与某个测试匹配的项目。
    • 和 map() 一样，给 filter() 赋予一个函数，它对数组中的每个项目调用该函数。
  • **join()**：与 split() 相反，将数组连接为字符串。
  • **toString()**：数组转换为字符串的另一种方法。
    • join() 可以指定不同的分隔符。
    • toString() 总是用逗号分隔。
