Chrome DevTools 学习笔记

2026/3/18 learn devtools

[toc]

📑 核心面板

今天大致学习了 Chrome DevTools，主要了解了以下四个 Tab：

  • Elements
  • Console
  • Sources
  • Network

🛠️ 技能掌握

1. Elements 面板

  • 查询 DOM 树：使用 Ctrl + F 快速查找节点。
  • 编辑样式：
    • 添加样式
    • 复制样式
    • 观察 Grid、Flex 布局

2. Console 面板 (Ctrl + Shift + J)

  • 执行语句：直接在控制台运行 JS 代码。
  • 实用快捷键/变量：
    • $_：返回上一条语句的执行结果。
    • $0：获取上一个选择的 DOM 节点（同理可用 $1, $2…）。
  • 常用方法：
    • console.log()
    • console.error(), console.warn()
    • console.table(), console.clear()
    • console.group(), console.time()

📅 后续计划

  • JavaScript 调试：将在学完 JS 基础后，进一步深入学习 Sources 面板的断点调试功能。
